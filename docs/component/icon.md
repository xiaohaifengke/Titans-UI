# Icon 图标
提供了一套常用的图标集合。

## 使用方法
设置 `icon` 属性为图标名称即可。
:::demo
```vue
<template>
  <div>
    <TiIcon icon="microphone"></TiIcon>
    <TiIcon icon="dish"></TiIcon>
    <TiIcon icon="circle-close"></TiIcon>
    <ti-button icon="search" icon-position="suffix" type="primary">插槽内容</ti-button>
  </div>
</template>
```
:::
