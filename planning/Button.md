# Button 的功规
- 基本用法：
    - 基础按钮（可通过props定制状态）
    - 支持插槽
    - 支持icon
    - 按钮组
    - 加载中 loading
    - disabled
    
    
- 接受的props:
    - type（按钮类型）：default/primary/success/warning/danger/info
    - mode（按钮模式）：fill/plain/outline/text
    - shape（按钮形状）：fillet/round/square
        > 当mode为text时，shape无效
    - size（按钮大小）：large/normal/small/mini
    - 图标按钮
    - 禁用状态
