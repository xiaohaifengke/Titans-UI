# Button 的功规
- 基本用法：
    - 基础按钮（可通过props定制状态）
    - 支持插槽
    - 支持icon
    - 按钮组 (额外的button-group组件)
      - 考虑添加的props:
        - disabled
    - 加载中 loading
    - disabled
    
    
- 接受的props:
    - label(和slot的功能类似)
    - type（按钮类型）：default/primary/success/warning/danger/info
    - mode（按钮模式）：fill/plain/outline/text
    - shape（按钮形状）：fillet/round/square
        > 当mode为text时，shape无效
    - size（按钮大小）：large/normal/small/mini
    - disabled
    - loading
    - icon
    - iconPosition
